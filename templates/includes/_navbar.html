Help

Parser: Select the correct parser for the command or file output you would like to convert to JSON or YAML. See the parser documentation for more information on each parser.

Command Output: Copy/Paste the command or file output you would like converted to JSON or YAML.

Pretty Print: If checked, formats the JSON result with indentation. Otherwise, the resulting JSON will be printed on a single line.

Raw Output: If checked, outputs pre-processed data.

YAML Output: If checked, outputs YAML instead of JSON.

About

Welcome to the jc demo web app!

jc is a command-line utility and Python library that converts the output of many commands and file-types to JSON or YAML for easier parsing in scripts.

jc is hand-crafted with love by Kelly Brazil.

This web app is based on jc version {{ jc_version }}.

For more information, please see the jc project github site.

The source code for this web app can be found here: jc-web source.
